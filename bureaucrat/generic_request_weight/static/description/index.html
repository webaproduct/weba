Generic Request Weight


Generic Request Weight is a module of the Generic Request application.

The module allows you to automatically sort requests by predefined rules depending on SLA, Priority, Kanban State, Request Type, etc.

In order to have access to these fields, user must belong to group "Request Manager".



You can specify the weight of request service


Additionally you can specify the weight of service level


You can specify the weight of request category


You can specify the weight of request type


You can specify the weight of request stage


As result, the weight of the request is determined by multiplying mentioned weights on following predefined weights:

  • Sla state weight
  • Activity state weight
  • Priority state weight
  • Kanban state weight
