Generic Resource Roles


The module extends the functionality of the Generic Resource application and provides the ability to organize access to resources by roles.

The main benefits of this approach are:

  • Explicit way of defining access. It is really easy to track who and when has/had access to particular resource.
  • Ability to grant access to resource for certain period of time
  • Universal approach, same for all resources and resource types
  • Ability to automatically grant access to related resources. For example, when you grant access to server for some user, you can easily configure system to automatically add access to all databases on this server for that user.

Create new resource and specify Resource Visibility to Restricted by Roles


Navigate to Roles page of resource and click Add new role link


Grant access to resource for specific person (or partner)


Take a look at list of all role links registered for this resource.
This way you can easily track who has access to this resource
