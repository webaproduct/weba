Task Timer

This module allows the user to start/stop the time of a task. Easy to calculate the duration of time taken for the task. You can start multiple tasks at the same time. We provide a default description option when the task ends. You can filter running tasks using filter!

Start or End Task from Kanban View.


Start Task from Form View, on click of 'Start Task' Button,Task Timer Started for that Task.


On click of 'End Task' Button pop up opens.


Enter Task Description, total Task Duration auto-calculated.


On click of 'End Task' Button inside Pop Up , Timesheet Entry Created inside Task.


Task Entry created into 'My Timesheet'.


Configuration for the multiple tasks and default description.


You can see running tasks using the filter.


Now we end that running task.


After the end task, the task description default comes based on configuration, and the total task duration auto-calculated.


Tasks list view.


You can start multiple tasks at the same time based on the configuration.


It shows an alert if you untick 'Multiple Task Allowed?' and try to start multiple tasks at the same time.
