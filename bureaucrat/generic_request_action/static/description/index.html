Generic Request Actions


Generic Request (Actions) is a module of the Generic Request application. It allows you to configure actions that will be automatically performed when processing a request.

  • You can select events to trigger actions
  • Specify routes on which actions are performed


Actions can do:

  • Server actions
  • Execute Python code
  • Assignments
  • Subscribe partners, channels, departments
  • Schedule activities
  • Send email using selected templates


You can use jinja2 templates in the fields to get field values.


Generic Request (Action Subrequest) is a module of the Generic Request application. It allows you to automatically create sub-requests using automated actions.

Set up automated action to create a subrequest with a specific configuration


Read the Generic Request Actions module guide for more information.
