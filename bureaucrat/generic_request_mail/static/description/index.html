Generic Request Mail


Generic Request Mail is a module of the Generic Request application.

It allows you to:

  • Set up email aliases and mail sources to catch email from desired adresses
  • Automatically create requests from incoming emails
  • Manually create requests in one click from any incoming email
  • Setup canned responses - short words that will be automatically substitute with configured text in chatters and messages

Configuration

While in the Requests app, click Configuration -> Mail Sources

Configure email aliases to catch emails from


Set up requests to be created using incoming emails


You can also manually create requests from any incoming email using Requests icon. Email body will be automatically inserted as a request text.


You can setup canned responses that will be used in chatters and messages. Configure special keywords; when you type these keyword, they can be automatically substituted with prepared text.
