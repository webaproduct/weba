Generic Request Related Document


Generic Request Related Document is a module of the Generic Request application. It allows you to link any Odoo documents with requests.

Set up related documents for each type of request


Select related documents when submitting the request



You can see if requests are related to documents from the kanban view


Read the Generic Request Related Doc module guide for more information.
