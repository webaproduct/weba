Generic Request Related Requests is a module of the Generic Request application. It allows you to establish links between requests.

Read the `Generic Request Related Requests <https://crnd.pro/doc-bureaucrat-itsm/11.0/en/Generic_Request_Related_Requests_admin_eng/>`__ Module Guide for more information.

This module is part of the Bureaucrat ITSM project.
