Generic Assignment Policy


Generic Assignment Policy is a technical addon. It allows you to create custom user assignment policies and rules by which these assignments will take place.

Key features:
  • Easily define rules to assign right user for any Odoo object
  • Custom assignment policies
  • Different policy models
  • Custom assignment rules for every policy
  • Conditional assignments
  • Assignments using Python expression
  • Assignments by User
  • Assignments by User Field
  • Assignments using another policy
  • Complex assignments using multiple related policies
  • Policy testing

You can add assignment types (e.g. HR related assignments, assignments by Team, Project assignments) by installing additional modules.
